Test Instructions:

All buttons should have consistent behavior:

  • Default: Transparent background, accent border and text
  • Hover: Accent background, contrasting text, slight lift
  • Active: Accent background, no lift, pressed shadow
  • Focus: Same as hover + visible focus ring
  • Disabled: Reduced opacity, no pointer interaction

Test all viewports: Resize window to test mobile (480px), tablet (768px), and desktop (1024px+)
